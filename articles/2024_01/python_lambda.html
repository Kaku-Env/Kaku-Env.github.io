lambda(ラムダ)式とは

lambdaは「無名関数」と呼ばれる、関数オブジェクトを生成することができる式のことです。

1行で関数が書けるため、戻り値や引数などに関数の戻り値を渡したい場合に、 あらかじめ関数を定義することなく、直接記述することができます。

文法

lambda式は以下のように書きます。

lambda 引数: 戻り値

これを関数で書くと、以下のようになります。


def func(引数):
    return 戻り値

ただし、lambdaは関数とは違い、関数名は名前空間に保存されません。
一時的に関数の戻り値を扱いたい場合に利用されます。

lambda式の具体的な使い方

例えば、以下のような引数を2倍した値を返すような簡単な関数を定義したい場合、lambda式なら 1行で書くことができます。 


# 関数の場合
def double(x):
    return x*2
print(double(3))

# lambdaの場合
double = lambda x: x*2
print(double(3))

# >> 6

しかし、このようにlambda式を変数に格納する使い方はあまり使われず、 PEP8でも非推奨とされています。

代入文を使うと、ラムダ式が提供できる唯一の利点(つまり、大きな式に埋め込めること)を消してしまいます。

PEP8: プログラミングに関する注意事項

lambda式は、生成した関数オブジェクトを、他の関数やメソッドの引数に直接渡して使うのが 基本的な使い方です。 

print(lambda x: x+1)
